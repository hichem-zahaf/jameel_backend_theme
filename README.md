# Jameel Backend Theme

A minimalist and elegant backend theme for Odoo 19 Community Edition. This theme transforms your Odoo backend with a clean design, sophisticated gold & grey color palette, and improved user experience.

## Features

### 1. Modified Structure for All Type Views

All views including Kanban, List, Form, and more have been redesigned with a modern, clean layout.

### 2. New Style for Active Elements

Active menus, radio buttons, and checkboxes have a fresh, modern design with improved usability.

### 3. New Color Combination

A sophisticated gold & grey color palette that brings elegance and professionalism to your backend.

### 4. New Look for All Applications

Every application gets a fresh, modern appearance that enhances the overall user experience.

### 5. Clean Layout and New Font Style

Improved typography and a cleaner layout that makes navigation and content consumption effortless.

### 6. Sidebar with New Menu Icons

A redesigned sidebar with new app icons and company logo for a more intuitive navigation experience.

## Installation

1. Place this module in your Odoo `addons` directory
2. Update the Odoo apps list:
   ```bash
   odoo-bin -c odoo.conf -d your_db -u all --stop-after-init
   ```
3. Install the module from the Apps menu or via CLI:
   ```bash
   odoo-bin -c odoo.conf -d your_db -i jameel_backend_theme --stop-after-init
   ```
4. Restart your Odoo server
