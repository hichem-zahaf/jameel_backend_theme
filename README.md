# Jameel Backend Theme

A minimalist and elegant backend theme for Odoo 19 Community Edition. This theme customizes the web client's appearance with a clean design, new color combinations, and improved UX.

## Features

- **Collapsible Sidebar** - Toggle sidebar with smooth animations
- **Modern Design** - Clean layout with updated color scheme
- **Fully Responsive** - Adapts to all screen sizes
- **Custom Icons** - Beautiful app menu icons
- **Enhanced Views** - Styled list, kanban, and form views

## Installation

1. Copy this module to your Odoo `addons` directory
2. Update the Odoo apps list
3. Install the module from the Apps menu

## Requirements

- Odoo 19 Community Edition
